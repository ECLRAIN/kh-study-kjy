## Test09

소수 판정 프로그램

소수는 나누어 떨어지는 값이 없는 숫자를 말합니다(ex : 2, 3, 5, 7, 11, 13, 17, 19, 23, ...)

어짜피 모든 숫자는 1로 나누기가 가능하고, 자기자신으로 나누기가 가능합니다

(ex) 4는 소수가 아닙니다
```
4 % 1 = 0
4 % 2 = 0
4 % 3 = 1
4 % 4 = 0
```
(ex) 5는 소수입니다
```
5 % 1 = 0
5 % 2 = 1
5 % 3 = 2
5 % 4 = 1
5 % 5 = 0
```
